# rfp-intelligence

	•	StageWon is a GenAI-powered platform that automates the end-to-end RFP-to-proposal pipeline for B2B sales teams — from RFP discovery to relevance scoring, team assembly, and proposal drafting.
	•	Uses LLM-based parsing and RAG to match RFPs against past credentials and CVs, cutting manual proposal drafting time from ~8 hours to under 45 minutes.
	•	Tech stack: React + FastAPI + Claude API, with a vector DB (ChromaDB) for retrieval and a CRM-style tracker for win/loss analytics — designed to be reusable across any consulting practice.
